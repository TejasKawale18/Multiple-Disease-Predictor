# 🏥 Multiple Disease Predictor

⭐ Predicts multiple diseases like Diabetes, Heart Disease, and Parkinson’s using Machine Learning in seconds

---

## 📌 Problem Statement

Early detection of diseases is critical, but many people lack access to quick preliminary diagnosis tools. Traditional diagnosis methods are time-consuming and require medical consultation.

---

## 🚀 Solution

This project provides a **machine learning-based web application** that:

* Predicts multiple diseases based on user input
* Provides instant results
* Helps in early awareness and decision-making

The system uses trained ML models to analyze medical parameters and generate predictions.

---

## ⚙️ Tech Stack

* Python
* Streamlit
* Scikit-learn
* Pandas, NumPy
* Machine Learning Models

---

## 🧠 Features

* 🩺 Predict **Diabetes**
* ❤️ Predict **Heart Disease**
* 🧠 Predict **Parkinson’s Disease**
* 📊 User-friendly interface
* ⚡ Real-time predictions

---

## 🧠 How It Works

1. User selects disease from sidebar
2. Inputs required medical parameters
3. ML model processes input data
4. System predicts disease likelihood
5. Result displayed instantly

---

## 📊 Models Used

* Logistic Regression
* Support Vector Machine (SVM)
* Decision Tree / Random Forest

(Models trained on medical datasets for accurate predictions)
